Presentación del Proyecto

Ontofrutis es una aplicación Web diseñado para hacer consultas SPARQL a una ontología dedicado a las frutas, es capaz de filtrar características específicas con un alto nivel de complejidad. Está desarrollada en C# y Angular 5, utilizando Protegé para crear la Ontología en OWL.
